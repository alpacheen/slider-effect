# slider-effect
A product page with a slider/blur effect
- effect works both <- to -> and vice versa
- "SEE MORE" section also with slider effect(one direction) with a "SEE ALL" to return back to initial section.

The goal is to create a slider effect with information about products for the consumer. Giving a brief description of the product in the initial view and the "SEE MORE" section providing more info.
